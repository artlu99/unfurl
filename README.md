A microservice to unfurl Open Graph data from URLs.

Endpoints:

GET: https://unfurl.artlu.workers.dev/?u=https://bcbhshow.com

health check:

GET: https://unfurl.artlu.workers.dev/health

bust cache:

POST: https://unfurl.artlu.workers.dev/bust

---

```
pnpm install
pnpm dev
```

```
pnpm run deploy
```
